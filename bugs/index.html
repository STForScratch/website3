Bug Bounty

Welcome to ScratchTools' Bug Bounty program. We take bugs very seriously, and sometimes so we're willing to do everything we can to stop them from reaching our users. If they do ever reach our users without our beta testers finding them, then we need the help of the users to let us know when there is one.


Security Bugs

Usually, beta testers never find security bugs because there probably aren't any. Sometimes, though, they do slip into the code. It's a race for our developers to get them out, because, if the wrong person knows about them, it could be bad. Please never report these publicly.
